ADPilatus
===========
An 
[EPICS](http://www.aps.anl.gov/epics/) 
[areaDetector](http://cars.uchicago.edu/software/epics/areaDetector.html) 
driver for the Pilatus pixel array detectors from 
[Dectris](http://www.dectris.com).
